# webpack-training

### How to run up this project
1. Install dependencies
    ```shell
    npm install 
    ```
2. Compiles and hot-reloads for development
    ```shell
    npm run start
    ```

3. Compiles and minifies for production
   ``` shell
   npm run build
   ```

4. Lints and fixes files
   ```shell
   npm run eslint
   ```

5. Type check
   ```shell
   npm run tsc
   ```
